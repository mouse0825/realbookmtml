個人書庫介紹

  • 個人化的書籍收藏空間，方便管理您的閱讀清單
  • 支援書籍分類和標籤功能，輕鬆整理您的藏書
  • 閱讀進度追蹤，記錄您的閱讀歷程
  • 個人閱讀筆記和心得分享功能
  • 書籍推薦系統，基於您的閱讀喜好推薦相關書籍

API 說明

使用的外部 API

API名稱 用途 說明
Google Books API 書籍資訊查詢 用於獲取書籍詳細信息，包括標題、作者、出版信息等
OpenLibrary API 書籍封面圖片 提供書籍封面圖片和預覽功能
GoodReads API 書評和評分 獲取書籍評論和評分信息
